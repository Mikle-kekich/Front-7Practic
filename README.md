Учебный проект серверного приложения на Node.js и Express. Реализует регистрацию пользователей, вход в систему с выдачей JWT-токена, а также CRUD-операции для управления товарами с защитой определенных маршрутов.


## 🛠 Технологии
* **Node.js** & **Express** — серверная логика и маршрутизация.
* **bcrypt** — хеширование паролей пользователей.
* **jsonwebtoken (JWT)** — генерация и проверка токенов авторизации.
* **crypto** — встроенный модуль для генерации уникальных ID.

*Примечание: В целях упрощения запуска проекта данные (пользователи и товары) хранятся в оперативной памяти (в массивах). При перезапуске сервера данные очищаются.*
